<b>Crypto Mesh Node (CMN) - Alpha</b><br><br>

Crypto Mesh Node is a peer-to-peer messaging and broadcasting application built for privacy, decentralization, and blockchain integration. It allows users to run nodes that connect directly to each other, send encrypted messages, and participate in the network while earning rewards.<br><br>

<b>Features:</b>
<ul>
<li>Encrypted peer-to-peer messaging</li>
<li>Broadcast messaging to all connected nodes</li>
<li>Auto-generated Binance Smart Chain (BSC) wallets for reward tracking</li>
<li>Update feed integration from GitHub</li>
<li>Earn 1 coin every 30 days for staying online (NOT READY)</li>
<li>Easy-to-use GUI with PyQt6 interface</li>
<li>Secure private key management and message signing</li>
</ul>

<b>Getting Started:</b>
<ol>
<li>Install and launch the Crypto Mesh Node executable.</li>
<li>Enter or load your node private key to connect.</li>
<li>Use the tabs to view logs, broadcasts, updates, wallet info, and About information.</li>
<li>Send messages to other nodes or broadcast to the network.</li>
<li>Keep your node running to earn hourly rewards.</li>
</ol>
